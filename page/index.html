UdonLibrary

  • 概要

    部活動で使用している C++ ライブラリです。香川県といえばうどんということで、UdonLibrary と名付けました。

  • アルゴリズム、通信、ユーザー定義型、シリアライザ、ドライバ等の機能を提供しています。 開発時間の短縮、品質向上、再利用性の向上のために制作しました。 ライブラリの技術水準を上げることで、部員のプログラミングスキル向上に繋がる考えており、実際にその効果が出ています。

  • CI/CD

    GoogleTest、ArduinoCli を使用して、ユニットテスト、構文テストを行い品質を保証しています。 Doxygen を使用してソースコードからドキュメントを自動生成しています。 ドキュメントページ

  • 使用技術

    C++ / Arduino / OpenSiv3D / Doxygen / GitHub Actions / GitHub Pages
