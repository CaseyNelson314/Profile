UdonLibrary

  • 概要

    部活動で使用している C++ ライブラリです。アルゴリズム、通信、ユーザー定義型、シリアライザ、ドライバ等の機能を提供しています。香川県といえばうどんということで、UdonLibrary と名付けました。

  • 開発経緯

    開発時間の短縮、品質向上、再利用性の向上のために制作しました。 ライブラリの技術水準を上げることで、部員のプログラミングスキル向上に繋がると考えており、実際にその効果が出ていると感じています。

  • オブジェクト指向

    オブジェクト指向の原則に則って設計を行っており、使用者が直感的に使用できるように設計しています。モーター等のデバイスは部品化されており、部品を組み合わせることでロボットのプログラムを簡単に作成できます。

  • 型安全な通信

    ROS に倣い、型を用いて通信を行っています。内部的にはシリアライザによってバイト列化され通信されます。 これにより、バイト列を直接扱うことなく通信を行えます。

  • CI/CD

    GitHub Actions を用いて CI/CD を行っています。 GoogleTest、ArduinoCli を使用して、ユニットテスト、構文テストを行い品質を保証しています。 Doxygen を使用してソースコードからドキュメントを自動生成しています。

  • 使用技術

    C++ / Arduino / Doxygen / GitHub Actions / GitHub Pages

EFSim

  • 概要

    三次元空間で電気力線のシミュレーションを行うWebアプリです。電荷を動かしたり、パラメーターの変更ができ、リアルタイムにシミュレーションを行えます。

  • 開発経緯

    電磁気学の理解を深めるために制作しました。結果としてかなり深く理解できました。この時初めて Web 系の技術に触れることができたので、一石二鳥でした。授業の教材として使用されています。

  • 使用技術

    Three.js / TypeScript / GitHub Actions / GitHub Pages

UniformPointVisualizer

  • 概要

    球面上に一様に分布された点の可視化 Web アプリです。

  • 開発経緯

    EFSim に用いられている 一般化螺旋集合アルゴリズムの挙動を確認するために制作しました。このアルゴリズムを用いて電気力線を出す方向を決定しています。

  • 使用技術

    Three.js / TypeScript / GitHub Actions / GitHub Pages

Visualizer

  • 概要

    ロボットに搭載されている紙飛行機の発射機構の操縦アプリです。操縦者側で使用し、目標となる台を選択すると、機構の角度などをロボットに送信します。 2022 年度全国高専ロボコン 四国地区大会、全国大会で使用しました。

  • 開発経緯

    目標の台の選択をコントローラーで行うのは難しいと感じたため、直感的に操作できるアプリを制作しました。OpenSiv3D を通じてオブジェクト指向の設計を学びました。

  • 使用技術

    OpenSiv3D / Arduino / C++
